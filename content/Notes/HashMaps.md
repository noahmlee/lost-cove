## HashMaps

A hash map is a data structure that stores key-value pairs and allows quick access, insertion and deletion of values using keys. A common analogy for hashmaps is a department store full of items. You could search each shelf looking at every item you come across until you find the item you are looking for, or you could go to the manager who knows exactly where your needed item is within the store. In this scenario the manager is using something similar to a hashmap to find your item.

Hashmaps are made up of key value pairs, and use a hash function to calculate an index into an array of buckets. The primary function is to use a key (a person's name), and find the value (the same person's email address).
